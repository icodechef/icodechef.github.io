Lime

一个 PHP 的微框架, 帮助你快速构建简单但强大的 RESTful 风格的网页应用和 API。它提供路由(routing)、依赖注入(dependency injection)、中间件(middleware)、视图继承、视图片段和安全 cookies 等功能。

Features

快速构建应用程序

Lime 能帮助您构建简单但强大的 RESTful 风格的网页应用和 API。支持路由(routing)、依赖注入(dependency injection)、中间件(middleware)、视图继承、视图片段等功能

轻内核、易扩展

核心的设计思路清晰、良好的架构，松散耦合，易于扩展维护。非常容易扩展和添加新的类库。

小巧而简单的框架

Lime 不打算做成一个大而全的框架。在保持简洁的同时提供足够强大的功能来提高你的生产力，在此基础之上允许您使用第三方扩展。

灵活的视图功能

使用 Lime 的视图继承功能可以为整个应用定义一种布局模板，也可以为每个操作定义不同的布局模板。而视图片段功能可以按需求将数据输出到视图中。
